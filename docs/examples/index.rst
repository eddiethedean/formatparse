Examples and Cookbook
=====================

Practical recipes for common parsing tasks. Each example is copy-pasteable with
``formatparse`` installed from PyPI.

**Topics covered in the cookbook:**

- Log lines and timestamps (:doc:`cookbook` — log parsing, URLs, config)
- Datetime fields — see :doc:`../user_guides/datetime_parsing`
- Validation pipelines — see :doc:`../user_guides/post_parse_validation` and
  :doc:`cookbook` (bidirectional validation snippet)
- Performance at scale — :doc:`../user_guides/performance` and ``compile`` /
  ``parse_batch`` in :doc:`cookbook`
